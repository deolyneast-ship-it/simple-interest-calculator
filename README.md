# simple-interest-calculator

Simple Interest Calculator
A lightweight, efficient script designed to calculate simple interest. This project serves as a foundational example of implementing mathematical logic within a software application.

#🚀 Overview

The Simple Interest Calculator is a tool that computes the interest earned or charged on a principal amount over a specific period at a fixed interest rate.

Key Features
Precision: Handles decimal inputs for interest rates and time.

Clarity: Provides a breakdown of the total amount (Principal + Interest).

Validation: Ensures inputs are numerical and logical.

#🧮 How it Works

The application utilizes the standard mathematical formula for simple interest:
                     $$I = \frac{P \times R \times T}{100}$$

Variable Definitions:

P (Principal): The initial amount of money.

R (Rate): The annual interest rate (percentage).

T (Time): The duration in years.
